Problem 3923. -- Jabby's Luckynumber

3923: Jabby's Luckynumber

Time Limit: 5 Sec  Memory Limit: 256 MB
Submit: 18  Solved: 13
[Submit][Status][Discuss]

Description

Jabby特别喜欢a,b两个数字，它把十进制下都仅由这两个数字组成的
数字成为幸运数。现在它得到了两个位数相同的幸运数l,r。它设l,r 之间的幸运数为a1,a2...������an现在它想知道∑A(i)*A(i-1),1<=i<=N-1,对1000000007取模的结果。

Input

第一行给定两个数a,b
第二行给出l
第三行给出r

Output

如题所述

Sample Input

4 7
474
777

Sample Output

2316330

HINT

ans=(474*477+477*744+744*747+747*774+774*777)%1000000007=2316330

1 <= a<b<=9

1 <= L<R<=10^100000
