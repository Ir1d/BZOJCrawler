Problem 2831. -- 小强的金字塔

2831: 小强的金字塔

Time Limit: 10 Sec  Memory Limit: 128 MB
Submit: 51  Solved: 24
[Submit][Status][Discuss]

Description

一个金字塔的区域可以用下面的不等式组来说明
L<=X<=R
O<Y<=X
0<B*Z<=A*x+c
其中L,R,A,C是非负整数，B是正整数.
现想知道，这个区域中的整数点 （x,y,z)有多少个

Input

五个整数A,B,C,L,R (L<=R)

Output

如题

Sample Input

2 3 1 1 3

Sample Output

9

HINT

输入的五个数不超过10^17,L<=R

Source

By 范浩强
