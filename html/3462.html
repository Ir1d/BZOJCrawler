Problem 3462. -- DZY Loves Math II

3462: DZY Loves Math II

Time Limit: 20 Sec  Memory Limit: 512 MB
Submit: 39  Solved: 16

Description

Input

第一行，两个正整数 S 和 q，q 表示询问数量。
接下来 q 行，每行一个正整数 n。

Output

输出共 q 行，分别为每个询问的答案。

Sample Input

30 3
9
29
1000000000000000000

Sample Output

0
9
450000036
