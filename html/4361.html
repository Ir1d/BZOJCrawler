Problem 4361. -- isn

4361: isn

Time Limit: 10 Sec  Memory Limit: 256 MB
Submit: 8  Solved: 8
[Submit][Status][Discuss]

Description

给出一个长度为n的序列A(A1,A2...AN)。如果序列A不是非降的，你必须从中删去一个数，
这一操作，直到A非降为止。求有多少种不同的操作方案，答案模10^9+7。

Input

第一行一个整数n。
接下来一行n个整数，描述A。

Output

一行一个整数，描述答案。

Sample Input

4
1 7 5 3

Sample Output

18

HINT

1<=N<=2000
