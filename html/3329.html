Description

Input

第一行一个正整数，表示数据组数据 ，接下来T行

Output

2*T行

第2*i行表示第i个数据中问题二的解，

Sample Input

1
1

Sample Output

1
2

HINT

不要mod 10^9+7


1<=N<=10^18

1<=T<=1000
