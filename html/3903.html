Problem 3903. -- 反垄断

3903: 反垄断

Time Limit: 40 Sec  Memory Limit: 512 MB
Submit: 47  Solved: 0
[Submit][Status][Discuss]

Description

Input

第一行为4个整数，分别为nA，nB，k，t。其中，k是航线的总数。接下来k行，每行两个
整数a，b，表示有一条航线连接A国a市和B国b市。

Output

第一行为一个整数，即目标式的最小值。

Sample Input

3 5 8 2
1 4
1 3
3 3
1 2
1 1
2 1
1 5
2 2

Sample Output

4

HINT

对于 100% 的数据，1 <= nA;nB;t <= 200;1 <= k <= 5000。
