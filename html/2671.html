Problem 2671. -- Calc

2671: Calc

Time Limit: 10 Sec  Memory Limit: 128 MB
Submit: 162  Solved: 86

Description

　　给出N，统计满足下面条件的数对(a,b)的个数：
　　1.1<=a<b<=N
　　2.a+b整除a*b
　

Input

　一行一个数N

　

Output

　一行一个数表示答案

Sample Input

15

Sample Output

4
