Problem 2456. -- mode

2456: mode

Time Limit: 1 Sec  Memory Limit: 1 MB
Submit: 2561  Solved: 1099
[Submit][Status][Discuss]

Description

给你一个n个数的数列，其中某个数出现了超过n div 2次即众数，请你找出那个数。

Input

第1行一个正整数n。
第2行n个正整数用空格隔开。

Output

    一行一个正整数表示那个众数。

Sample Input

5
3 2 3 1 3

Sample Output

3

HINT

100%的数据，n<=500000，数列中每个数<=maxlongint。
