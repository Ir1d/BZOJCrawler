Problem 2785. -- [东莞选拔2012]树的计数

2785: [东莞选拔2012]树的计数

Time Limit: 10 Sec  Memory Limit: 128 MB
Submit: 10  Solved: 0
[Submit][Status][Discuss]

Description

给出两个整数n和d，求出有n个节点并且两个节点间最长距离为d的标号树的个数。
标号树即是树上每个结点都标有一个不同的编号。

Input

每行包含两个整数n和d。
每个文件包含多行，读入直到文件结束。

Output

 
对于每一行，输出一行包含一个整数。

Sample Input

2 1
3 1
3 2
4 3

Sample Output

1
0
3
12

HINT

对于所有数据，1≤n≤50，0≤d≤n。
